## Setup & Run Locally

1. Clone the repo  
   ```bash
   git clone https://github.com/samara-x/crowdfunding-frontend.git
   cd crowdfunding-frontend
2. Install dependencies: `npm install`
3. Create `.env` file in root: VITE_API_URL=https://fundingfourcrowds-0aa4332162ae.herokuapp.com
4. Start development server: `npm run dev`
5. Open the localhost port shown
